# 📊 Business Intelligence & Finance Dashboard

An interactive data analytics application designed to turn raw financial ledgers into actionable business insights. This project demonstrates professional-grade data visualization and rapid web deployment using the Python ecosystem.

## 🚀 Key Features
* **Dynamic KPI Tracking**: Real-time calculation of Total Budget, Actual Spend, and Variance.
* **Interactive Visualizations**: Grouped bar charts and donut charts for categorical expense breakdown.
* **Responsive Design**: Fully optimized for both desktop and mobile viewing via Streamlit.
* **Developer Portal**: Integrated sidebar with professional links and tech stack overview.

## 🛠️ Tech Stack
* **Language**: Python 3.x
* **Framework**: [Streamlit](https://streamlit.io/)
* **Data Handling**: [Pandas](https://pandas.pydata.org/)
* **Visualization**: [Plotly Express](https://plotly.com/python/plotly-express/)
